Sitio web Andrea Fernández

Tecnología:
- Proyecto desarrollado en React (Vite)

Para correr el proyecto:
1. Instalar dependencias:
   npm install

2. Ejecutar en local:
   npm run dev

3. Build de producción:
   npm run build

Estructura:
- src/: componentes y secciones
- public/: assets públicos

Notas:
- El sitio ya está funcional
- Incluye animaciones, flip cards y sistema de idioma
- Puede integrarse un campus virtual como nueva sección o sistema externo

Producción:
El sitio está desplegado y funcionando en Netlify.
